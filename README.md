# Sokoban

A simple Sokoban game, made by Godot 4.

## Requirements

* Godot 4.2.1

## Level format

| Level Element | Character |
| --- | --- |
| Floor | `.` |
| Wall  | `#` |
| Player | `@` |
| Player on goal square | `A` |
| Box | `X` |
| Box on goal square | `C` |
| Goal | `O` |
