# Sokoban

Sokoban, made by Godot 4.

## Level format

| Level Element | Character |
| --- | --- |
| Wall  | `#` |
| Floor | `.` |
| Player | `@` |
| Player on goal square | `A` |
| Box | `X` |
| Box on goal square | `C` |
| Goal | `O` |
